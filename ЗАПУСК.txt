📋 БЫСТРЫЙ СТАРТ (3 минуты) - Теперь ещё проще!
════════════════════════════════════════════════════════════════

1️⃣ УСТАНОВИТЕ ЗАВИСИМОСТИ

   npm install


2️⃣ ИНИЦИАЛИЗИРУЙТЕ БАЗУ ДАННЫХ (SQLite - ничего устанавливать не нужно!)

   npm run db:init

   База данных SQLite будет автоматически создана в файле:
   database/teplo_arctici.db


3️⃣ ЗАПОЛНИТЕ ТЕСТОВЫМИ ДАННЫМИ (опционально)

   npm run db:seed

   Это создаст:
   - 3 благотворительных сбора
   - 20 пожертвований
   - 5 отчетов о расходах


4️⃣ ЗАПУСТИТЕ СЕРВЕР

   npm run dev

   Сервер запустится на http://localhost:3000

🔧 ВОЗМОЖНЫЕ ПРОБЛЕМЫ
═══════════════════════════════════════════════════════════════

❌ Ошибка компиляции sqlite3

   РЕШЕНИЕ (Windows):
   1. Установите Visual Studio Build Tools:
      https://visualstudio.microsoft.com/downloads/
      (выберите "Build Tools for Visual Studio")
   2. При установке выберите "Desktop development with C++"
   3. После установки запустите: npm install

   АЛЬТЕРНАТИВА (Windows):
   npm install --build-from-source

   РЕШЕНИЕ (Linux/Mac):
   sudo apt install build-essential  # Ubuntu/Debian
   # или
   xcode-select --install  # macOS

❌ Порт 3000 уже занят

   РЕШЕНИЕ: Измените порт в файле .env:
   PORT=3001

❌ Нужно пересоздать базу данных

   РЕШЕНИЕ:
   1. Удалите файл: database/teplo_arctici.db
   2. Запустите: npm run db:init && npm run db:seed
